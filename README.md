## 중점을 둔 부분

1. 필드와 레코드, 폼이 각각의 데이터가 아닌 연결되어 있는 부분이 있다고 판단하여 세 데이터에 공통적으로 사용할 수 있는 타입과 로직을 만드려 노력했습니다
2. 최대한 antd의 기본 디자인과 기능(Table,Form)을 사용했습니다
3. 컴포넌트 추상화를 진행하면서, 단일 책임 원칙을 고려했습니다
